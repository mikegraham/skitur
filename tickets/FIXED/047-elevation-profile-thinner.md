# 047: Make Elevation Profile a Little Thinner
**Status:** FIXED
**Priority:** P2

## Description
The elevation profile chart line/markers are a bit thick. Make them a little thinner for a cleaner look.

## Resolution
Reduced line widths and marker sizes in the elevation profile chart:
- Background trace line width: 6 → 4
- Colored line traces width: 6 → 4
- Hover marker size: 12 → 8 (~33% reduction)

This creates a cleaner, less heavy visual appearance while maintaining readability.
